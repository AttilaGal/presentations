What is Rust?

  • started in 2006 as a side project
  • MIT license
  • sponsored by Mozilla since 2009
  • "most loved programming language" (Stack Overflow 2016 & 2017)

System's programming language

  • fine grained control over allocations
  • no required garbage-collector
  • minimal-runtime
  • "close to the metal"

Prevents almost all crashes

  • safe by default
  • sophosticated type system and analysis
  • no segfaults
  • no null pointers
  • no dangling pointers
Segfaults are caused by a program trying to read or write an illegal memory location.

Eliminates data races

  • ownership guarantees
  • borrowing prevents dangling pointers
  • strong, safe abstractions
Ownership: variables are moved to new locations, preventing the previous location form using it. There is only ever ONE owner of data!
